# JVM(자바가상머신), 개발환경 구축

# 1. JVM

- JAVA Virtual Machine, 자바 애플리케이션을 실행시키기 위한 가상 컴퓨터 환경
- os에 종속받지 않고 cpu가 java를 인식, 실행할 수 있게 하는 가상컴퓨터이다.
- [Hello.java](http://Hello.java) 자바프로그램 → 컴파일러 → [Hello.](http://Hello.Java)Class 바이트코드 → 윈도우용, 리눅스용, 맥용 JVM → 각 실행파일 → 각 pc
